# SmartPhone Unity - WEEK 01
『　TAP DODGE　』


# I.ゲーム内容　────────────────────

Player(〇)を障害物(□)に当たらないようにレーン移動して避けるゲーム
生存時間がSCOREになる


# II.操作　──────────────────────

Playerより左をタップすれば左のレーンに移動
Playerより右をタップすれば右のレーンに移動


# III.難易度が上昇　──────────────────

SCORE(生存時間)が大きくなるほど、障害物の速度が上昇していきます
